# Amortizaciones Web App

Aplicación web con autenticación para registrar estudiantes, crear amortizaciones y permitir que el profesor revise los envíos.

## Instalación

1. Abre una terminal en la carpeta `Nueva carpeta\amortizaciones-app`.
2. Ejecuta:

```bash
npm install
```

## Uso

```bash
npm start
```

Luego abre `http://localhost:3000` en tu navegador.

## Acceso

- Profesor:
  - Correo: `profesor@escuela.com`
  - Contraseña: `Profesor123`

- Estudiantes: pueden registrarse desde el formulario de registro.

## Funcionalidades

- Registro de estudiantes con login.
- Generación de amortizaciones con cálculo automático.
- Guardado de registros asociados al alumno.
- Revisión de amortizaciones por parte del profesor.
- Vista de alumnos registrados y envíos de amortización.
